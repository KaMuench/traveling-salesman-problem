# traveling-salesman-problem (IntellijIDEA Project)
This was a small project from the data structurs and algortithms module of my university. 
The task was to build a [genetic algorithm](https://en.wikipedia.org/wiki/Genetic_algorithm) to find good solutions for an [TSP](https://de.wikipedia.org/wiki/Problem_des_Handlungsreisenden). A genetic algorithm might not be the most efficient approach to
find good solutions for a TSP, but it does the job.
